This project recreates the classic Pac-Man gameplay where the player navigates through a maze, collects pellets, and avoids ghosts. The game includes smooth animations, collision detection, score tracking, and multiple difficulty levels.

🚀 Features
🎮 Classic Pac-Man gameplay experience
🧠 Intelligent ghost movement with basic AI logic
⚡ Smooth animations using requestAnimationFrame
🟡 Pellet & power-pellet system with scoring
❤️ Lives system and game-over/win states
🎚️ Difficulty levels (Easy, Medium, Hard) affecting ghost speed
🎨 Responsive UI with modern styling
⌨️ Keyboard controls (Arrow keys / WASD)
🔁 Replay option after game ends
🛠️ Tech Stack
HTML5
Structure of the game and UI screens (landing page & game canvas)
CSS3
Styling, layout, animations, and responsive design
JavaScript (Vanilla JS)
Game logic, player movement, ghost AI, collision detection
Rendering using Canvas API
Game loop using requestAnimationFrame
🎯 Concepts Used
Computer Graphics (Canvas Rendering)
Game Loop & Frame Updates
Collision Detection
Grid-based Movement System
Basic AI (Ghost Path Decision Logic)
Event Handling (Keyboard Input)
State Management (Score, Lives, Game Status)
▶️ How to Run
Clone the repository
Open index.html in your browser
Select difficulty level
Start playing 🎮
